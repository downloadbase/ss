Who is this class for?

This class is designed for those who are interested in learning how to use a computer. Even if
you have never used a computer before, and especially if you are willing to invest in yourself.
You will find the class particularly useful if:

  • If you want to know how to use a computer,
  • If you work in a place where computer usage
    is a requirement,
  • If you want to better find your way around in this computer world and more.
  • Including basics of computer languages [HTML & CSS]
